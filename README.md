# Mini Vocab Trainer

Small single-page app to add vocabulary and practice quick multiple-choice quizzes. Built in one day.

## Features
- Add / delete / edit vocabulary (word, translation, example)
- Practice mode with randomized multiple-choice questions
- Persistent storage via `localStorage`
- Responsive, simple UI
